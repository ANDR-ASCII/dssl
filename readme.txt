По самой программе:

Добавление/удаление шарика по клику правой кнопки мыши.
Перемещение по клику левой кнопки мыши.
Запускается/останавливается расчет по нажатию кнопки.

Сборка:

В качестве IDE использовалась VS 2017 15.2 v141.

Для генерации solution'a необходимо создать переменную окружения:

CMAKE_PREFIX_PATH = [some path]\Qt\Qt.5.7.0\5.7\msvc2015_64

Также, чтобы при запуске скомпилированного процесса находились все .dll и .lib зависимости
необходимо в переменную окружения PATH добавить следующие пути:

[some path]\Qt\Qt.5.7.0\5.7\msvc2015_64\plugins
[some path]\Qt\Qt.5.7.0\5.7\msvc2015_64\plugins\platforms
[some path]\Qt\Qt.5.7.0\5.7\msvc2015_64\bin

Для запуска приложения, необходимо сделать стартовым проект app, для тестов
calculationtests соответственно.

PS Если возникнут проблемы с генерацией solution, пришлю готовый.
